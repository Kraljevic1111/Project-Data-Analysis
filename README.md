# 📊 Project: Data Analysis with Pandas

## 📌 Overview

This project demonstrates basic data analysis using the Python library **pandas**.
The dataset represents a simple set of customer orders, including products, prices, and quantities.

The goal is to calculate key business insights such as:

* Total revenue
* Customer spending
* Best-selling products
* Average prices

---

## 📁 Dataset Description

The dataset is created manually and includes the following columns:

| Column   | Description               |
| -------- | ------------------------- |
| order_id | Unique ID of the order    |
| customer | Name of the customer      |
| product  | Product purchased         |
| price    | Price per unit            |
| quantity | Number of units purchased |

---

## ⚙️ Features / Analysis Performed

### 1. Total Price per Order

A new column `total_price` is created:

```
total_price = price * quantity
```

---

### 2. Total Spending per Customer

Calculates how much each customer spent in total.

---

### 3. Top Customer

Finds the customer with the highest total spending.

---

### 4. Total Revenue per Product

Calculates total earnings generated by each product.

---

### 5. Revenue Sorted (Descending)

Sorts products based on total revenue (highest first).

---

### 6. Best-Selling Product (by Quantity)

Finds which product was sold the most (by number of units).

---

### 7. Average Price per Product

Calculates the average price for each product.

---
